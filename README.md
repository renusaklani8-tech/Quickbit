Quickbit – Stop chasing cleaners, start booking instantly

Quickbit is a full-stack on-demand home services web application that connects users with verified service professionals like cleaners, plumbers, electricians, and more.

It provides a seamless platform for instant booking, real-time management, safety features, and admin control.

🛠️ Tech Stack
Frontend: HTML5, CSS3, Vanilla JavaScript
Backend: Node.js, Express.js
Database: MySQL
✨ Key Features
🔐 Role-based Authentication (Users, Workers, Admin)
📅 Real-time Booking Management
📲 OTP Verification System
🆘 SOS Emergency Alert with Location Tracking
👩‍🦰 Women Safety Features
🧑‍🔧 Verified Service Professionals
📊 Admin Dashboard for Full Control
⚡ Fast and responsive UI
